# Goose & Potatoes Quiz App

This is my Streamlit app for our class project. I coded it in VS Code, pushed it to GitHub, and deployed it on Streamlit Cloud. You can try it here: [goose-and-potatoes.streamlit.app](https://goose-and-potatoes.streamlit.app).

---

## Purpose
The app is basically a silly quiz, but the main point is to show that I can use a bunch of Streamlit components in one project.

---

## Inputs
I used different widgets to collect answers:
- Radio buttons (pizza fork question)
- Checkbox (WiFi scream)
- Slider (potato rating)
- Text input (weird names)
- Multiselect (spirit animal)
- Number input (sock count)
- Date input (homework pain day)
- Time input (reality crisis)
- Color picker (mood color)
- Text area (goose battle plan)

---

## Outputs
The app shows results using:
- Dataframe (answers table)
- Download button (CSV export)
- Metrics (questions answered, goose threat level)
- Progress bar
- Success/warning/error/info messages
- Map (RTU Pasig)
- Video (goose clip)
- Expander (hidden goose strategy)

---

## Components
I made sure to include more than 20 Streamlit components: text elements, inputs, outputs, buttons, and status messages.
