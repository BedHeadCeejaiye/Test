# Goose & Potatoes Quiz App

This is my Streamlit app for our assignment. I built it in VS Code, uploaded it to GitHub, and deployed it on Streamlit Cloud. The app is live here: [goose-and-potatoes.streamlit.app](https://goose-and-potatoes.streamlit.app).

---

## 🎯 Purpose
The app shows how different Streamlit components can be used together in one project.  
It’s set up as a fun quiz with random questions, but the main goal is to demonstrate inputs, outputs, and layout.

---

## 👥 Target Users
- My professor and classmates (for grading).  
- Anyone curious about Streamlit and wants to try a silly interactive quiz.

---

## 📥 Inputs
I used several input widgets to collect answers:
- Radio buttons (pizza fork question)  
- Checkbox (WiFi scream)  
- Slider (potato rating)  
- Text input (weird names)  
- Multiselect (spirit animal)  
- Number input (sock count)  
- Date input (homework pain day)  
- Time input (reality crisis)  
- Color picker (mood color)  
- Text area (goose battle plan)

---

## 📤 Outputs
The app displays results using:
- Dataframe (answers table)  
- Download button (save CSV)  
- Metrics (questions answered, goose threat level)  
- Progress bar  
- Status messages (success, warning, error, info)  
- Map (RTU Pasig)  
- Video (goose clip)  
- Expander (hidden goose strategy)  
- Captions and headers for structure

---

## 🛠️ Components Checklist
I included more than 20 Streamlit components, covering:
- Text elements (title, header, caption, write)  
- Input widgets (radio, checkbox, slider, text_input, multiselect, number_input, date_input, time_input, color_picker, text_area)  
- Output elements (dataframe, download_button, metric, progress, map, video, expander)  
- Buttons and status messages
